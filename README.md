Detector de Vazamentos com ESP32  
Projeto desenvolvido para a Faculdade Mackenzie usando IoT.  

Funcionalidades  
- Monitoramento em tempo real de umidade e fluxo de água  
- Alertas via MQTT e acionamento de relé
